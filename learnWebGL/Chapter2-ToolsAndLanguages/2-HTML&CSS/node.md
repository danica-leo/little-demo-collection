# 2.2 - HTML and CSS

一个基础网页是由HTML和CSS构成的，HTML描述网页的内容，而CSS描述页面的样式。
如果页面代码写的较为细心，那同一个页面中的相同HTML，基于特殊的CSS实现，不同设备上可以展示完全不同的样式
在本教程中不需要成为专业的Web开发者，我们的学习重心是 Computer graphics ，所以掌握一些基础内容就好了。
